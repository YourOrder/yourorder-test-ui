# YourOrder Demo UI

Демонстрационный веб-интерфейс сервиса YourOrder на моковых данных.

Внутри:

- витрина товаров;
- корзина и оформление заказа;
- список заказов со статусами;
- кабинет продавца;
- компании, товары и остатки;
- имитация ключевых событий: `company.created`, `product.created`, `order.created`, `stock.reserved`.

Запуск:

```bash
npm start
```

Открыть:

```text
http://127.0.0.1:5173
```
